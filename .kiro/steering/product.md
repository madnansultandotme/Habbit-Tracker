# Product Overview

Habit Tracker is a web application for tracking daily habits and monitoring progress over time.

## Core Features

- Create and manage personal habits
- Mark habits as complete for each day
- Track streaks and completion rates
- View monthly progress via calendar
- Light/dark theme support
- Data persisted to localStorage (frontend) with MongoDB backend available

## User Experience

- Single-page dashboard with tabs for habits list and calendar view
- Stats cards showing total habits, daily progress, and completion rate
- Toast notifications for user feedback
- Responsive design for mobile and desktop
